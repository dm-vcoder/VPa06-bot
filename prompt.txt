Создай проект на Python:

Telegram-бот для командной работы, для добавления задач в общий список, используй aiogram для создания tg бота.
Интерфейс: бот принимает команды (/start, /add, /list, /list_csv).
Логика: обработка сообщений, добавление задач, вывод списка, вывод списка файлом в формате CSV.
Данные: хранит SQLite3 (таблица tasks: id, text, user, created_at).

Минимальный рабочий пример:

/start - приветствует.
/add - сохраняет задачу в базу.
/list - выводит все задачи в красивом и удобно читаемом виде, содержащем id / дата и время создания задачи / описание задачи, где id это трёхзначный номер.
/list_csv - отправляет CSV-файл с задачами пользователю  (разделители данных в файле = табуляция) .
/del - предлагает выбрать задачу для удаления (выбор задачи = ввод её id), и удаляет задачу после отправки её id

Сокращай рассуждения, выполняй процессы в фоновом режиме и выдавай конечный результат по факту его готовности.
